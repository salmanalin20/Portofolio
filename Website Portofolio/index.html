Halo! Saya seorang Biologist

Sebagai mahasiswa biologi dengan passion di bioinformatika dan visualisasi data, saya menggabungkan ilmu biologi dengan teknologi untuk menganalisis data genomik dan biologis.

Saya berspesialisasi dalam analisis bioinformatika, visualisasi data biologis, dan penelitian laboratorium molekuler. Setiap proyek penelitian adalah kesempatan untuk menemukan insight baru dari data.

Ketika tidak di laboratorium atau menganalisis data, saya suka membuat konten edukasi sains, mengeksplorasi tools bioinformatika terbaru, dan berbagi pengetahuan dengan komunitas peneliti.
